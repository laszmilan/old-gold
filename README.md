# Old Gold

The website for **Old Gold**, an OSR fantasy adventure game of daring exploration, creative problem-solving, and fast-paced tactical combat. It's a static site (plain HTML, CSS, and vanilla JavaScript with no build step) featuring the home page, an online rulebook rendered from Markdown, and a set of random generators, all available in English and Hungarian.

Published by [Tiny Raven Press](https://tinyravenpress.itch.io/).
